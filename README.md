## Failure_Prediction_Tapaculo

```
1. Collect data using linux shell script from tapaculo365.
2. Send data to hadoop.
3. Find when user's machine stop or fail.
4. Analyzing data using hadoop.
5. Find correlation and pattern about machine fail and time and data.
6. So, Tapaculo administrator can be predict that the machine may have problems at certain times or by certain data.
7. As a result, system administrator can be provide guide line to user about user's machine.
```
